# AlphabeticalSchema

This gem sorts the columns in your schema.rb file alphabetically to prevent conflicts.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'alphabetical_schema'
```

## Usage

Run `rake db:migrate`.
